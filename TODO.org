;; rumorboy.io
;;
;; Plug-in за GoogleChrome/Firefox, който добавя p2p чат върху всички страници,
;; които посещава потребителя. Идеята е да имаш опцията да чатиш анонимно с
;; всички хора, които четат това, което и ти и това да не се контролира от
;; никого с цел цензура или измама.
;;
;; Пример: Влизаш в сайт, който продава измемен продукт и там има само някакви
;; псевдо коментари от доволни клиенти, активираш приложението и разбираш че
;; това е измама, като си говориш с реални хора.
;;
;; За HackFMI ще направим само plugin за Chrome и примерен node сървър, който ще
;; прави първоначалната връзка между хората. Очаквам да не ни отнеме почвече от
;; 1 ден. : )
;;
;; Приложение за хром и за вграждане в сайт, където до новини, сайтове,
;; материали се показва един бутон и хората могат да коментират аномнимно, p2p
;; съдържанието. Така например за сайтовете, които са измама ще може да се
;; добави това и веднага да сигнализра на хората, какво става, на нереалните
;; новини също. За да работи това трябва да има поне един човек, който да е
;; online и така всички следващи ще получават всички коментари. Хората ще дават
;; и общ рейтинг на прочетения шит.
;;
;; Уникално за домейна. Vote for it or switch to it.
;; Споделяне на натоварването по online хората.
;;
;; Open page, initiate chat on page.
;;
;; http://blog.mgechev.com/2014/09/03/webrtc-peer-to-peer-chat-with-react/
;;
;; One host and all connect to it. And it spreads data. If it fails other
;; becomes the host.
;;

first -> server -> you are host
second -> server -> send id to host
third -> server -> send id to host
host leaves
forth -> server -> you are host
second -> claims host (with date joined) -> now you are host
       -> inform outdated host and get his and all his connections and they just
          logged in
